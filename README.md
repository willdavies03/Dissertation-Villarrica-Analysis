# Dissertation-Villarrica-Analysis

This repository contains all code and datasets used in the analysis of TROPOMI SO2 emisions from Villarrica volcano (2018-2026), as presented in the accompanying dissertation. The materials provided here allow full reproduction of the figures, processing steps and analytical workflow. 

# Respository Content
# Jupyter Notebooks
SO2Analysis_LinearInterpolatedData
The primary analysis notebook containing the full processing workflow and majoirty of figures used in the dissertation. 

Villarrica_example_analysis
A supplementary notebook containing additional example analyses and figures used to support interpretation and methodological explanation

# Data Files
Villarrica_2018-06-01_2026-03-01_SO2_time_seriesnanmean.csv
A CSV file containing the SO2 time series derived from the datacube

Villarrica_2018-06-01_2026-03-01_so2_data.nc
The Level-3 SO2 data cube (NetCDF format). This file forms the core dataset for the study.
